#  AI: Multimodal Health Intelligence System

##  Overview

Codecure AI is an intelligent health-tech system that integrates symptom analysis and emotion recognition to provide early-stage disease prediction and health insights. The system leverages AI to enhance preventive healthcare and improve patient awareness.

---

## 🚀 Features

* 🧠 Text-based emotion detection using BERT
* 👁️ Facial emotion recognition using CNN
* 💊 Disease prediction based on symptoms
* 🩺 Interactive symptom selection UI
* 📊 Real-time health analysis

---

## 🛠 Tech Stack

* Python
* Streamlit
* PyTorch
* Scikit-learn
* Transformers (Hugging Face)
* OpenCV

---

## ⚙️ Installation

```bash
git clone <your-repo-link>
cd health_ai_project
pip install -r requirements.txt
```

---

## ▶️ Run the App

```bash
streamlit run app.py
```

---

## 🧠 Technical Workflow

1. User inputs symptoms (text + selection)
2. NLP model analyzes emotional tone
3. CNN model detects facial emotion
4. Machine learning model predicts disease
5. System outputs results with insights
